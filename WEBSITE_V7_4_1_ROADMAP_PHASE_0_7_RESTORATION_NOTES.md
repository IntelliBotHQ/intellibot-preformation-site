## Update purpose

V7.4.1 restores the detailed public Roadmap phase structure from Phase 0 through Phase 7 while preserving the professional timeline/card presentation.

## Main change

The Roadmap page no longer compresses the project into only four broad roadmap cards. It now shows eight phase-gated public roadmap phases:

- Phase 0 — Governance, architecture, and clean foundation
- Phase 1 — Local foundation, credentials, persistence, and diagnostics
- Phase 2 — Market data, indicators, screener, strategy contracts, and watchdogs
- Phase 3 — Risk engine, sandbox broker, order lifecycle, and canonical backtester
- Phase 4 — Live execution hardening after deterministic simulation
- Phase 5 — Operator cockpit UI and progressive unlock workflow
- Phase 6 — Advisory context, alerts, deployment packaging, and operator support
- Phase 7 — Commercialization, beta, compliance, and funding-pathway review
