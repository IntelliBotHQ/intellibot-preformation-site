- Reworked homepage copy and layout.
- Reworked Product page copy and operating-model presentation.
- Upgraded Interface page with click-to-enlarge gallery behavior.
- Reworked Roadmap page into a phase-gated timeline.
- Reworked Trust page into safety-architecture cards.
- Reworked Disclosures page into disclosure cards instead of a simple table.
- Reworked Contact page to clarify acceptable and unacceptable contact use.
- Added `assets/site-interactions.js` for the local interface lightbox.
- Added CSS microinteractions, gallery modal styles, control-stack cards, timeline cards, and disclosure cards.
- Corrected embedded logo treatment inside the seven concept UI images using the approved IntelliBot emblem source.
