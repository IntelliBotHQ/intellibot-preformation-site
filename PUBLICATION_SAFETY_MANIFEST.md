# Publication Safety Manifest

## Package purpose

This package is prepared for direct upload to a **public GitHub Pages repository** as the interim IntelliBot informational site.

## Publication-safe contents

| Category | Included? | Assessment |
| --- | ---: | --- |
| Public HTML website pages | Yes | Intended for publication |
| Public styling and navigation JavaScript | Yes | Intended for publication |
| Emblem-only public brand image | Yes | Does not display a corporate suffix |
| Interim public Terms / Privacy / Disclosures content | Yes | Informational, not final adopted corporate policies |
| GitHub Pages `CNAME` and `.nojekyll` files | Yes | Required/deployment-appropriate |
| Public deployment instructions | Yes | Contains no secrets |

## Explicitly excluded

| Category | Included? |
| --- | ---: |
| Vercel deployment configuration | No |
| Internal TPMS files | No |
| Investor letter, brochure, SAFE or financing materials | No |
| Capital-raising terms or investor intake | No |
| Controlled beta agreement | No |
| Internal security/incident-response plan | No |
| Analytics integrations or tracking pixels | No |
| Beta waitlist forms | No |
| API keys, passwords, tokens, or private credentials | No |
| Branding stating an incorporated legal entity | No |

## Language verification

The public pages state that IntelliBot is a pre-formation product initiative in development, do not represent a released trading product, do not solicit an investment, do not claim performance, and do not accept beta enrollment at this interim stage.
