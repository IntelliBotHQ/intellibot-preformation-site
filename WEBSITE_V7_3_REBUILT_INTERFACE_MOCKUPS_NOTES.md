# IntelliBot Website V7.3 — Rebuilt Interface Mockups Notes

## Purpose

V7.3 replaces the prior patched PNG interface mockups with fully rebuilt SVG interface mockups.

## Defects addressed

- Upper-left logo appeared malformed, misaligned, or wrapped in a visible patch rectangle.
- Enlarged modal images could still show old or inconsistent logo treatment.
- First gallery image could render larger than the other six.
- Mockup footer language was inconsistent across images.
- Some mockups contained alternate shield/check logo marks that did not match the approved IntelliBot emblem.

## Fix

- Created seven rebuilt SVG mockup assets.
- Embedded the approved IntelliBot emblem artwork directly into each SVG.
- Used the same top-left brand block in every image.
- Removed alternate shield/check logo marks from the rebuilt design.
- Used identical footer language in every image.
- Updated both thumbnail and click-to-enlarge modal references to SVG assets with `?v=7.3.0`.
- Preserved prototype image placement on `/interface/` only.
