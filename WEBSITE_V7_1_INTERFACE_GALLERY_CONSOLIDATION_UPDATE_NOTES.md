# IntelliBot Website V7.1 — Interface Gallery Consolidation Update

## Baseline

Derived from:

`IntelliBot_Website_V7_Robust_Concept_UI_GitHubPages_PublicRepo.zip`

## Update purpose

V7.1 consolidates all prototype/user-interface imagery into the dedicated `/interface/` page.

## Why this update was made

After the interface page was added, the homepage and Product page still repeated multiple prototype interface images. That created unnecessary visual redundancy and made the website feel overloaded.

## Public-site changes

- Removed prototype interface images from the homepage.
- Removed prototype interface images from the Product page.
- Kept all prototype interface images on `/interface/`.
- Added text-only calls to action from the homepage and Product page to the Interface Gallery.
- Preserved the correct Website V7 logo and CSS assets.
- Preserved the existing publication-safety boundary.
